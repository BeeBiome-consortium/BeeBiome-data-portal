BeeBiome API

https://beebiome.org/beebiome/

BeeBiome is a resource providing a comprehensive catalogue of currently available sequence datasets about bee-associated microbes and virus.

This BeeBiome API facilitates access to the BeeBiome data.

Data retrieved through the API is available under the Creative Commons Zero license (CC0) from Switzerland promoting ease of use in scientific and bioinformatics projects.

Methods to retrieve statistics

GET /statistics/all
Retrieve data statistics about the current BeeBiome release

Methods to retrieve entries

GET /sample/all
Retrieve all Beebiome entries
GET /sample/{acc}
Retrieve Beebiome entries with a BioSample accession containing {acc}
POST /sample/advanced-search
Retrieve Beebiome entries filtered by parameters provided in the request.
Parameters can be combined. In that case, filters are applied using 'AND' boolean logic. Example: https://beebiome.org/beebiome/sample/advanced-search?host=mellifera&organism=Lactobacillus will return entries for which the host is mellifera and the organism is Lactobacillus.
You can use the following parameters:
ParameterField to which the parameter value must matchExample
bioSampleAcc BioSample accession https://beebiome.org/beebiome/sample/advanced-search?bioSampleAcc=SAMN07311030
bioProjectAcc BioProject accession https://beebiome.org/beebiome/sample/advanced-search?bioProjectAcc=PRJNA392822
host Scientific name of the natural host (as opposed to laboratory) to the organism from which the sample was obtained https://beebiome.org/beebiome/sample/advanced-search?host=mellifera
organism Scientific name of the organism that harbours the host https://beebiome.org/beebiome/sample/advanced-search?organism=Lactobacillus
collectionDate Date on which the sample was collected https://beebiome.org/beebiome/sample/advanced-search?collectionDate=2014-12-14
geoLocationName Geographical location name where the sample was collected https://beebiome.org/beebiome/sample/advanced-search?geoLocationName=Switzerland
librarySources Type of source material that is being sequenced https://beebiome.org/beebiome/sample/advanced-search?librarySources=GENOMIC
libraryLayouts Library layouts (SINGLE or PAIRED end reads) https://beebiome.org/beebiome/sample/advanced-search?libraryLayouts=PAIRED
libraryStrategies Sequencing technique intended for the library https://beebiome.org/beebiome/sample/advanced-search?libraryStrategies=WGS
instruments Sequencing instrument for library https://beebiome.org/beebiome/sample/advanced-search?instruments=Illumina
centerName Organization name involved into the project https://beebiome.org/beebiome/sample/advanced-search?centerName=University of Lausanne
biosamplePackageAcc BioSample package accession (more details) https://beebiome.org/beebiome/sample/advanced-search?biosamplePackageAcc=MIGS.ba.host-associated
biosamplePackageName BioSample package name (more details) https://beebiome.org/beebiome/sample/advanced-search?biosamplePackageName=bacteria/archaea
